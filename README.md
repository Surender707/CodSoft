# CodSoft Internship Projects

A collection of Machine Learning projects completed during the CodSoft internship program.

## Projects

### 1. Credit Card Fraud Detection
Detects fraudulent credit card transactions using machine learning classification techniques.
- **Tech:** Python, Scikit-learn, Pandas
- **Files:** `Credit Card Fraud Detection/`

### 2. Iris Flower Classification
Classifies Iris flowers into three species based on sepal and petal measurements.
- **Tech:** Python, Scikit-learn, Pandas
- **Files:** `Iris Flower/`

### 3. Movie Rating Prediction
Predicts movie ratings using features from IMDb India dataset.
- **Tech:** Python, Scikit-learn, Pandas
- **Files:** `Movie/`

### 4. Titanic Survival Prediction
Predicts passenger survival on the Titanic using classification models.
- **Tech:** Python, Scikit-learn, Pandas
- **Files:** `Titanic/`

## Getting Started

```bash
pip install -r requirements.txt
python train.py   # Train the model
python main.py    # Run the application
```
